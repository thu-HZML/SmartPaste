## 1. 产品简介与界面交互

**SmartPaste** 是一款结合了**桌宠**交互形式的**智能剪贴板管理工具**。它不仅能帮您记录历史复制内容，还集成了 AI 处理、OCR 识别及云端同步功能，是您办公效率的得力助手。

### 1.1 桌宠交互 

应用启动后，会在桌面右下角显示一个动态的**桌宠**形象

- **动态反馈**：桌宠会根据您的操作做出反应（如眨眼、环顾四周），视线会跟随鼠标移动，并在您移动鼠标、敲击键盘时模拟动作
- **唤出主菜单**：**单击**桌宠即可弹出主菜单，由此可进入用户信息、历史记录、收藏夹、云端及设置页面
- **快捷操作**：可用快捷键呼出/隐藏（见设置页面），桌宠是所有功能的集合入口，建议保持开启

## 2. 剪贴板使用（可离线）

### 2.1 自动捕获

无需任何操作，应用会在后台持续运行。您在任意软件（浏览器、文档、聊天软件）中执行**“复制”**操作，**SmartPaste** 会自动记录。

- **支持格式**：纯文本、图片、文件等

### 2.2 呼出与粘贴

- **呼出界面**：点击主菜单中按钮或按下自定义快捷键（默认为快捷键设置中配置的键位），即可打开历史记录列表
- **单条复制**：在列表中找到目标记录，点击右侧的 **“复制”图标**，内容即回到系统剪贴板，可直接粘贴
- **批量复制**：按住 **SHIFT 键** 并单击多条记录，可进行多选复制，一次性粘贴多条内容

## 3. 数据管理与OCR（可离线）

### 3.1 数据编辑与整理

- **编辑**：点击记录卡片右上角**“编辑”**图标，可修改文本内容
- **备注**：点击记录卡片右上角**“备注”**图标，为记录添加自定义说明，方便记忆
- **删除**：
  - **单删**：点击单条记录卡片右上角**“删除”**图标
  - **批量清空**：点击历史记录页面右上角**“删除”**图标。支持“一键清空非收藏记录”或“一键清空所有记录”（需要在设置中选择）
  - **自动清理**：系统会根据设置（如保留30天）自动删除过期数据

### 3.2 收藏与分组

- **收藏**：单击**“收藏”**图标存入“全部”收藏夹；**双击**图标可选择具体分类收藏夹（支持新建收藏夹）。点击已收藏历史记录可取消收藏于某收藏夹。在收藏界面可新建收藏夹并访问收藏夹
- **智能分组**：系统会自动识别内容类型并分组，您可在顶部栏选择查看不同分组的内容

### 3.3 搜索与 OCR 识别

- **全能搜索**：在搜索框输入关键词，支持搜索文本内容、图片OCR识别内容、文件名、文件路径名。还可以在顶部栏右侧日历设置时间范围进行筛选
- **图片 OCR**：对于复制的图片，系统会自动进行 OCR 文字识别。您不仅可以复制识别后的文字，还可以通过搜索图片中的文字来找到该张图片

### 3.4 离线备份（导入/导出）

- **导出**：进入 **设置 -> 数据备份**，点击导出，系统将生成包含所有本地数据的压缩包
- **导入**：在新设备或重装后，选择导入该压缩包，即可覆盖恢复本地数据

## 4. 账号管理

### 4.1 注册与登录

- **注册**：在主菜单点击头像或用户名栏可跳转到用户信息页面，在此选择“注册账户”，输入用户名、邮箱及密码即可。
- **登录**：使用注册信息登录后，可开启云端同步和 AI 功能

### 4.2 个人中心

- **信息维护**：点击菜单栏的头像或用户名进入用户主页，可修改头像、个人简介及密码
- **注销/删除**：支持退出当前账号或永久删除账户（注意：删除账户将清空云端数据）

## 5. AI Agent 智能助手

### 5.1 启用 AI

进入 **设置 -> AI Agent 设置**，**在登录状态下**，开启“启用 AI 助手”开关，可以选择AI服务，我们提供的”默认“服务可选择的模型有 DeepSeek 等五个；用户也可以自行设置AI服务，即选择默认以外的AI服务（如 OpenAI/Google等），并填写相应的API密钥/base_url等（注：此资源需要用户自行获取）。

### 5.2 AI 功能使用

复制一段内容后，桌宠上方会自动弹出 AI 助手悬浮窗，提供以下功能：

- **提问**：基于刚复制的内容向 AI 提问
- **总结**：快速生成长文本的摘要
- **翻译**：将外文内容直接翻译为中文
- **搜索**：基于关键词进行联网搜索，并提供相关网页链接

## 6. 云端同步与安全

### 6.1 云端同步

- **开启**：登录后，进入 **设置 -> 云端入口**，打开“启用云端同步”
- **同步机制**：支持“自动同步”（可设置频率）和“手动同步”（点击立即同步按钮）
- **多端一致**：您的历史记录和个性化配置（如快捷键、个性化配置）会自动在所有登录设备间保持一致

### 6.2 数据安全 (E2EE)

- **端到端加密**：在云端设置中开启此功能
- **效果**：开启后，所有数据在上传前即在本地加密。服务器端无法解密您的任何文件或文本，确保隐私绝对安全

### 6.3 隐私过滤

- **敏感词过滤**：进入 **设置 -> 安全与隐私**，开启过滤功能并勾选类型（身份证、银行卡、手机号等）。复制时，系统会自动识别敏感信息并标记（也会显示在历史记录中，只是内部标记）
- **隐私记录**：被标记为隐私的内容**不会上传至云端**，可在**设置 -> 安全与隐私**页面的“查看隐私记录”入口统一查看

## 7. 个性化设置

您可以点击主菜单的“设置”图标，对应用进行全方位定制：

| **设置板块**   | **可配置项**                                                 |
| -------------- | ------------------------------------------------------------ |
| **通用设置**   | 开机自启、托盘图标显示、历史记录保留时间                     |
| **快捷键**     | 主窗口呼出、剪贴板页面呼出、一键清空历史等                   |
| **剪贴板参数** | 最大记录数限制、忽略短文本（如<3字符）、忽略大文件、删除确认开关、自动排序开关 |
| **安全与隐私** | 敏感词过滤规则、隐私记录管理                                 |
| **数据备份**   | 更改存储路径、离线数据导入/导出                              |

## 8. 常用操作流程索引

### 流程 A：日常高效办公（复制粘贴）

1. 在文档/网页中连续复制多段图文内容
2. 按下 **快捷键** 唤出剪贴板界面
3. 对信息进行“收藏”、“备注”、“编辑”、“OCR”、“删除”、“搜索”等管理操作
4. 对需要的记录进行复制；或按住 SHIFT 多选，一次性复制所有内容

### 流程 B：跨设备数据迁移（离线版）

1. **旧设备**：点击桌宠 -> 设置 -> 数据备份 -> **导出数据** -> 获取压缩包
2. **传输**：通过U盘或局域网将压缩包传至新设备
3. **新设备**：点击桌宠 -> 设置 -> 数据备份 -> **导入数据** -> 选择压缩包 -> 完成覆盖

### 流程 C：跨设备数据同步（云端版）

1. **设备 A**：登录账号，确保“云端同步”已开启，点击“立即同步”确保数据上传
2. **设备 B**：登录同一账号
3. **自动合并**：应用会自动拉取云端最新数据与配置，实现无缝接力

### 流程 D：AI 辅助阅读/翻译

1. 复制一段外文或长难文本
2. 观察桌宠上方的 AI 悬浮窗
3. 点击 **“翻译”** 或 **“总结”**
4. 查看 AI 生成的结果，并可直接复制结果使用
